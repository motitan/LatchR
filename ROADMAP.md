# Roadmap

## Now (v0.x hardening)

- Stabilize project create/save/open flows
- Improve timeline smoothness and precision UX
- Batch operations for selected events
- Better error messages for import/export failures
- Clip export diagnostics (per-clip success/failure summary)

## Next (v1.0 target)

- Undo/redo stack for timeline and event-list edits
- Schema versioning and migration for `.latchr.json`
- Robust autosave recovery on startup
- Bulk label/tag editing tools
- Better keyboard shortcut system and shortcut editor

## Later

- Plugin-style import/export adapters
- Multi-track timelines and custom track layouts
- Team-shared project packs
- Optional cloud sync (self-hosted first)

## Contributions wanted

- Testing infrastructure
- Performance optimization for large timelines
- Documentation and tutorials
- UI/UX refinements for analyst workflows
